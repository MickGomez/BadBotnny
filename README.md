# BadBotnny
Sensor Infrarrojo E18-d80nk posee logica negada. Al detectar un objeto manda un 0 (LOW)
Calibrado a 60 cm, da un 0 (LOW) para distancias mayores a 60cm.
Logica inversa en todas las lineas de codigo
